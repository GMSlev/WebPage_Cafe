# ☕ Originne - Café Especial

A **Originne** é uma landing page desenvolvida para transmitir a sofisticação e o cuidado artesanal envolvidos na produção de cafés especiais. O projeto foca em um design limpo, tipografia elegante e uma paleta de cores terrosa.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando apenas tecnologias web fundamentais:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização avançada, incluindo:
    * **Flexbox & Grid:** Para alinhamentos precisos e layouts modernos.
    * **CSS Variables (Root):** Para uma paleta de cores organizada e fácil manutenção.
    * **Google Fonts:** Utilização das fontes *Montserrat* e *Raleway*.
    * **Position Sticky:** Para um cabeçalho fixo que melhora a navegação.

## ✨ Funcionalidades

- [x] Seção Hero com card de boas-vindas e imagem estilizada.
- [x] Seção "Sobre" com foco em narrativa de marca (Storytelling).
- [x] Grade de diferenciais (Features) com efeitos de hover interativos.
- [x] Scroll suave entre as seções da página.
